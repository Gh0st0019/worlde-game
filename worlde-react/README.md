# Worlde Pixel Italiano

Gioco in stile pixel basato sulla logica di `worlde.py`: indovina la parola italiana lettera per lettera con 6 tentativi.

## Avvio

```bash
npm install
npm run dev
```

## Struttura

- `src/App.jsx`: logica e UI del gioco
- `src/data/wordBank.js`: parole originali del file Python
- `src/App.css` e `src/index.css`: stile pixel art
